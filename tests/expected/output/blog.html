Poole has basic blog support. If an input page's file name has a structure like page-title.YYYY-MM-DD.post-title.md, e.g. blog.2010-02-27.read_this.md, Poole recognizes the date and post title and sets them as attributes of the page. These attributes can then be used to generate a list of blog posts:

  • I ate all the pokemans - March 01, 2010
  • Doctors in my penguin - February 22, 2010

Have a look into input/blog.md to see how it works. Feel free to adjust it to your needs.
